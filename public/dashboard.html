Source Catalog

Populates the country list the migration engine reads from source_catalog. Run this on a fresh database before starting a migration, or whenever a new country goes live upstream. This does not affect migration runs, checkpoints, or import errors — those are separate operational state, untouched by a catalog refresh.

Reset Everything

Wipes every migration run, checkpoint, and import error, plus the field mapping and source catalog tables — then immediately re-seeds field mappings from the committed dataset and refreshes the source catalog from the live Leads DB, in this same action. You do not need to click "Refresh Catalog" afterward. This is irreversible: there is no undo and no backup. Any field mappings you have edited on the Field Mappings page are replaced, not merged — your edits will be lost along with everything else.
